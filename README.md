# Korean Learning App

Приложение для изучения корейского языка с системой учёта прогресса учеников.

## Установка

1. Установить зависимости:
```bash
pip install -r requirements.txt
```

2. Запустить приложение:
```bash
python app.py
```

3. Открыть в браузере:
```
http://localhost:5000
```
